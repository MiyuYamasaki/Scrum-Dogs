ONLY NEED TO DO ONCE:
Open terminal/command window
change directory to where you want folder to be created
run cmd: git config --global user.email "(your email)"
run cmd: git config --global user.name "(your username)"
run cmd: git clone https://github.com/miyuyamasaki/scrum-dogs.git

DO EVERYTIME:
change directory to new folder
git pull to receive current updates
move any new files you want to add into the current folder on your computer
run cmd: git add "(File Name)"
run cmd: git commit -m "(Description of File being added)"
run cmd: git push origin master
